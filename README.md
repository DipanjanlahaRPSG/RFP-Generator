# RFP Generator - AI-Powered Platform

An AI-powered RFP (Request for Proposal) generation system that reduces RFP creation time from 8-12 hours to under 1 hour. Built for RPSG group companies (CESC, CPDL, MPSL).

## Overview

The RFP Generator uses a three-source content generation architecture:
- **NEW Sections (6)**: Generated by LLM from user input
- **OLD Sections (8)**: Retrieved via RAG from historical RFP corpus
- **RULES Sections (11)**: Fixed templates for legal/compliance

## Features

### Frontend (Next.js)
- Conversational chat interface with adaptive questions
- Section review with approval workflow
- AI evaluation metrics panel
- Real-time regeneration with version tracking
- Word document export

### Backend (FastAPI + Python)
- RAG-powered document retrieval using ChromaDB
- OpenAI GPT-4o for LLM generation
- Three-source architecture for RFP sections
- API endpoints for analyze, generate, regenerate, export
- SQLite database for session management

## Quick Start

### Prerequisites
- Node.js 18+ and npm
- Python 3.10+
- OpenAI API key

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at: http://localhost:3000

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
cp .env.example .env  # Add your OPENAI_API_KEY
python server.py
```

Backend API runs at: http://localhost:8000

### Environment Variables

Create a `.env` file in the `backend` directory:

```env
OPENAI_API_KEY=your_openai_api_key_here
OPENAI_MODEL=gpt-4o
```

## API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/analyze` | POST | Analyze initial request, generate questions |
| `/api/questions` | POST | Get next question in flow |
| `/api/generate` | POST | Generate all 25 RFP sections |
| `/api/regenerate` | POST | Regenerate single section |
| `/api/export` | GET | Export RFP as Word document |
